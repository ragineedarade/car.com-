 
# CAR.COM

## Features

- **Responsive Design**: The website is designed to be responsive and looks great on all devices.
- **Interactive Elements**: Hover effects on images and interactive buttons enhance the user experience.
- **Search Functionality**: Integrated Google search functionality to allow users to search for information.
- **Chat Widget**: A floating WhatsApp chat widget for user inquiries and support.
- **Marquee Slider**: A marquee slider to display important announcements and contact information.

## Technologies Used

- **HTML5**: For the structure of the website.
- **CSS3**: For styling and layout.
- **JavaScript**: For interactivity and dynamic content.
- **Font Awesome**: For icons used in the navigation and social links.
- **Ionicons**: For additional icons in the footer.

## Setup Instructions

1. **Clone the Repository**: Clone this repository to your local machine using the following command:
    ```bash
    git clone https://github.com/ragineedarade/car-com.git
    ```
2. **Navigate to the Directory**: Change to the project directory:
    ```bash
    cd car-com
    ```
3. **Open in Browser**: Open `index.html` in your preferred web browser to view the website.

## Usage

- **Home Page**: Start by exploring the home page (`index.html`) which provides links to various sections such as About, Contact, and different car models.
- **About Page**: Learn more about the website and its purpose.
- **Contact Page**: Get in touch with the website owner through the contact information provided.
